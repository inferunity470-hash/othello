# ビッド式オセロ ルール仕様書

## 概要

通常のオセロ (リバーシ) に「**入札による着手権の取り合い**」を加えた変則ルール。
両プレイヤーは初期チップを持ち、毎ターン秘密入札で着手権を競う。

## 用語

| 用語 | 意味 |
|---|---|
| 先手権トークン | 同額入札時にどちらが勝つかを決める仮想トークン。初期は黒が保持 |
| 入札 | 0 以上、自身の所持チップ数以下の整数 |
| 落札 | 入札の高い方が着手権を得る |
| ホルダー | 先手権トークンの保持者 |

## 1 ターンの流れ

```
[BIDDING]
  両者が秘密に入札 (0 ≤ bid ≤ 持ちチップ)
   ↓
[RESOLVING]
  高い方が落札。同額ならホルダーが落札
   ↓
[PLACING]
  落札者が任意の合法手で着手 (オセロ通常ルール)
   ↓
着手後、ホルダーが置いた場合のみ先手権トークンが移動
   ↓
次ターン (BIDDING に戻る)
```

## チップ支払い (3 種の競売方式)

### 🪙 ファースト (First-Price)
- 落札者が **自分の入札額** を支払う
- 敗者は支払いなし
- 例: A=20, B=10 → A 落札、A が 20 失う

### 🎲 セカンド (Second-Price / Vickrey)
- 落札者が **相手の入札額** を支払う
- 敗者は支払いなし
- 例: A=20, B=10 → A 落札、A が 10 失う
- 数学的に「真の評価額を入札する」のが弱支配戦略

### 💸 オールペイ (All-Pay)
- 落札の有無に関わらず **両者が自分の入札額を失う**
- 例: A=20, B=10 → A 落札、A が 20 失い、B も 10 失う
- 落とすと丸損なので、入札はメリハリが重要

## 特殊フェーズ

### FREE_MOVE
- 一方の合法手がゼロのとき、もう一方が **無償で着手** できる (チップ消費なし)

### FINAL_MOVE
- 旧仕様: 両者 0 チップでホルダーが 1 手プレイ
- **現仕様**: 廃止 (下記 ENDED のルールに統合)

### ENDED
- 両者に合法手がない、または両者がチップを使い切った時点で終了
- **入札の結果両者が 0 チップになった場合、そのターンの落札者は着手してから終了する** (支払った分の手は指せる)
- 着手後はチップ切れのため、無償着手 (FREE_MOVE) も発生しない

## 連続 0 入札制限

- 同一ターン内ではなく、両者連続 N ターン 0 入札したら強制最小入札 1
- 設定で制限ターン数を変更、または無制限にできる

## 勝敗判定

1. 終了時の **石数が多い方** が勝ち
2. 同数の場合は **残チップが多い方**
3. それも同数なら引き分け

## ハンディキャップ

- 黒・白で初期チップ数を別々に設定可能
- 弱い側に多めのチップを与えてバランス調整できる

## 戦略要素

- **入札金額の駆け引き**: 価値の高い局面でいくらまで払うか
- **トークンの維持/手放し**: ホルダーは同額入札で勝てる代わりに、着手するとトークンを失う
- **チップの温存**: 序盤に使い切ると終盤に身動きが取れない
- **エクスプロイト防御**: 「初手 50 / 次ターン all-in」のような単純戦略は AI 側で対策済み

## オンライン対戦の特殊ルール

- **再戦**: 対局終了後、両者同意で同じ部屋で再戦可能 (色入替)
- **観戦**: ルームには 2 人 + 観戦者 (任意人数) が参加可能
- **入札の秘匿**: サーバ側で物理的に保証 (相手の入札は revealed まで HIDDEN)
